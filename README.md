# Employee Exit & Asset Clearance Management System

A full-stack HR management platform to handle employee resignations, department clearances, asset returns, and exit analytics.

## 🚀 Tech Stack

### Frontend
- React.js
- React Router DOM
- CSS

### Backend
- Node.js
- Express.js

### Database
- SQLite

---

## ✨ Features

- Employee resignation submission
- Clearance approval workflow
- Asset return management
- Exit analytics dashboard
- Clearance history tracking
- Asset history tracking
- REST API integration
- Responsive sidebar dashboard
- SQLite database storage

---

```bash
backend/
frontend/
```

## 🔗 API Endpoints

### Resignation
- POST `/api/resignations`

### Clearance
- POST `/api/clearance`
- GET `/api/clearance`

### Assets
- POST `/api/assets`
- GET `/api/assets`

### Analytics
- GET `/api/dashboard/exits`

## ⚙️ Installation

### Clone Repository

```bash
git clone <repo-link>
```

### Backend Setup

```bash
cd backend
npm install
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```
